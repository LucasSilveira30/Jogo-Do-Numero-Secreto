# Jogo do Número Secreto

Minijogo de adivinhar um número secreto sorteado. Conforme você vai adivinhando, o programa te dá dicas, o que permite que você chegue mais próximo de descobrir o resultado a cada tentativa.

A atividade em si foi a elaboração do código em JavaScript para o funcionamento do jogo (arquivo "app.js"). 
Esse foi o projeto final do curso "Lógica de programação: mergulhe em programação com JavaScript", disponibilizado pela plataforma "Alura".
